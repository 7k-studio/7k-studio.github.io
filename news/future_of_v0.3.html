The version 0.3 will grey out, but the future will bring some colors

07.04.2026 v0.3

Current state of Deadalus

It has been quite a fascinating time. On October 2025, I presented the first usable version of the Deadalus program: v0.3. For the first time it allowed for the absolute minimum required to design a 3D wing or 2D profile. With a basic tools and widgets it was possible to create a project. An interactive OpenGL viewport replaced MapPlotLib in Airfoil Designer and a viewport in Wing Designer works now more like industry standard.

Grey future of version 0.3

It's hard for me to tell how wide range of an audience I have gathered since I do not collect any data. Therefore I am not sure if this message will find the target group of v0.3 users. As a solo developer it takes to much time for me to work on 2 projects at once. I'd really like to add more features to the program but the way I created the v0.3 makes it painful to scale up the project. Many things are hard-coded, many functions are duplicated or executed pointlessly, 2D spline is defined the other way than 3D and so on... Therefore I announce that version v0.3.4 will be the last update to current release.

Deadalus true colours

It doesn't mean I kill the project - it is the other way around. I work very hard on the new version v0.4 with:

  • Stability improvement
    • The way classes communicate in the program has been changed drastically to ensure better performance
  • Improvements to old widgets and bunch of new ones
    • Mostly visible in Airfoil Designer but Wing Designer will also undergo changes
  • New functionality in Airfoil Designer
  • Extended level of control when creating an airfoil

It is hard to announce any realise date yet as there are still many things I have to learn to provide you with the best experience and ensure everything works as intended.
